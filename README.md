## 下载

### Docker

```
docker pull apachecn0/tutorialgateway-bi-zh
docker run -tid -p <port>:80 apachecn0/tutorialgateway-bi-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g tutorialgateway-bi-zh
tutorialgateway-bi-zh <port>
# 访问 http://localhost:{port} 查看文档
```
